# 🏋️ Gym Tracker

A console-based Gym Management Application built with Java and MySQL.

## 📌 About the Project
This application helps users track their gym workouts efficiently.
Users can manage exercises, create workout sessions, and log 
their sets, reps, and weight — all stored in a MySQL database.

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL
- Eclipse / VS Code

## ✨ Features
- Manage Users
- Manage Exercises
- Create Workout Sessions
- Log Sets, Reps and Weight for each Exercise
- Save and Retrieve all data from MySQL Database
- Interactive Menu-driven System

## ▶️ How to Run
1. Clone the repository
2. Set up MySQL database
3. Update DB credentials in the config file
4. Run Main.java
